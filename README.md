Current foundation:

- Angular frontend application
- NestJS backend API foundation
- Astro/Starlight documentation site
- PostgreSQL database planning
- Docker-based local development
- Documentation as code
- Conventional Commits and repository workflow standards
- Planned AI/RAG capabilities introduced incrementally
